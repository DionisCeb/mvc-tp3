Il s'agit d'un projet de réservation de voitures. Les tables de la base de données sont décrites dans le fichier car_rent.sql. 
La page "home" est la page sur laquelle arrive le client dès qu'il ouvre le projet "/" ou "/home". Pour vous connecter, vous devez choisir l'option de vous connecter avec l'un des utilisateurs de car_rent.sql
Pour effectuer une réservation, appuyez sur le bouton de réservation qui vous amène à une section d'introduction, 
suivi d'un bouton qui vous amène à la page de création de réservation. Une fois la réservation créée, l'utilisateur est dirigé vers la page principale de réservation, 
où toute réservation peut être modifiée ou supprimée.

Cette application dynamique envoie la réservation par email et dans la section /bookings vous pouvez voir dans chaque réservation que vous pouvez générer la version pdf. Pour cette utilisation : "composer require phpmailer/phpmailer", "composer require dompdf/dompdf ".
